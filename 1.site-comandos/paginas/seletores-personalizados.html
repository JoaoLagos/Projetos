SELETORES PERSONALIZADOS (ID e CLASS)

ID e CLASS:

ID: id= "[nome]" é ÚNICO

CLASS: class= "[nome]" pode ser VÁRIOS

ID (Identificador) (#):

  • HTML:

    No HTML, para criar um identificador na tag, dentro da tag, use: id= "[nome]"

    OBS1: Não é legal colocar o mesmo id em outras tags. Para isso, use class ="[nome]".

    OBS2: id: único; class: pode vários.

  • CSS:

    E no CSS, use: [tag]#[id] {}

    OBS: Se fizer só #[id] {}, qualquer coisa que tiver o [id] será modificado.

CLASS (Classe) (.):

  • HTML:

    No HTML, para criar uma classe na tag, dentro da tag, use: class= "[nome]"

    OBS1: Se quiser, pode a mesma class em várias tags diferentes.

    OBS2: id: único; class: pode vários.

  • CSS:

    NO CSS, use: [tag].[class] {}

    OBS: Se fizer só .[class] {}, qualquer coisa que tiver o [class] será modificado.

OBS: Posso usar várias class em uma mesma tag. Exemplo: <h2 class= "[nome1] [nome2]">.

Pseudo-Classes (:):

Para colocar uma pseudo-classe, use: [tag]:[informação] {}

OBS:Para mexer em alguma tag dentro de outra tag, pode-se usar: [tag] > [tag] {}

Pseudo-Elementos (::):

Para colocar um pseudo-elemento, use: [tag]::[informação] {}

Tag Filho (>):

Para mexer em alguma tag dentro de outra tag, pode-se usar: [tag] > [tag] {}
