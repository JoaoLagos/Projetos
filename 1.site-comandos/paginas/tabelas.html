TABELAS

  • Criação de uma tabela:

    Para criar uma tabela, use: <table>

    Dentro da tabela possui hierarquia como visto abaixo:

    • <table> (PEQUENA)
      • <tr> (Table Row)
        • <th> (Table Header)
        • <td> (Table Data) -> O conteúdo, informação

    • <table> (GRANDE), mas se quiser usar em pequena, pode também
      • <thead>
        • tr, th, td
      • <tbody>
        • tr, th, td
      • <tfoot>
        • tr, th, td

  • Colar os dados da tabela (um do lado do outro):

    Por padrão, os dados vem separados. Para colar os dados da tabela, no CSS dentro de table {}, use: border-collapse: [parâmetro];

    Parâmetros:

    • separate: Default
    • collapse: Junta (cola), um do lado do outro, na tebela

  • Alinhamento dos dados da tabela:

    Para alinhar horizontalmente é fácil, só usar text-align: [parâmetro];. Mas para alinhar verticalmente, use: vertical-align: [parâmetro];

  • Caption (Legenda) e Escopo de Títulos da tabela:

    Para ter um título na tabela, use: <caption>

    Para referenciar um título (dizer ao que ele se refere), use: <th scope="[parâmetro]">

    Parâmetros scope:

    • col: Referência da coluna
    • row: Referência da linha
    • colgroup: Referência do grupo de colunas, ou seja, caso haja um título mesclado e precise referenciá-lo para um grupo de colunas (mais de uma coluna),.
    • rowgroup: Referência do grupo de linhas, ou seja, caso haja um título mesclado e precise referenciá-lo para um grupo de linhas (mais de uma linha).

  • Efeito Zebrado nas Tabelas:

    Para criar o efeito zebrado, deve-se usar a pseudo-classe nth-child em tr> Para isso, use: tr:nth-child(#n){}

  • Cabeçalho Fixo:

    Para isso, primeiro a tabela precisa ter posicionamento relativo, ou seja, position: relative;, depois coloque os títulos do cabeçalho em posição sticky, position; sticky; onde ele vai ficar, como top: 0;, e o background, pois essa informação é perdida.

  • Mesclagem de Células:

    Para expandir um célula, na linha ou na coluna, para que ela venha ocupar mais de um espaço, ###span="[quantidade]"

    Parâmetros:

    • colspan="[quantidade]": Se for expandir para coluna
    • rowspan="[quantidade]": Se for expandir para linha

  • Tabelas Responsivas

    Para ter uma tabela responsiva que se adapte a tela, pode colocar a tabela dentro de uma div, por exemplo: <div><table></table></div>

    E, depois, colocar as configurações no CSS da div referente à tabela.

    Uma configuração boa para se usar nesse caso é o overflow: [parâmetros];

    Parâmetros overflow:

    • auto: Ele faz a rolagem, MAS só do elemento div (e não da página inteira).
    • scroll: Parecido com o auto.
