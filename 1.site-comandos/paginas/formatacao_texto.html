FORMATAÇÃO DE TEXTO


Organização de Títulos:

Para título de grau 1 (prioridade máxima), use: <h1>

Para título de grau 2 (prioridade 2), use: <h2>

E assim por diante...

Parágrafo:

Para criar parágrafos, use: <p> [Texto] </p>

Estilo dos Textos:

  • Negrito:
    • Para colocar em negrito, use: <b> (NÃO SEMANTICO)
    • Para colocar em negrito, use: <strong> (SEMANTICO)

  • Itálico:
    • Para colocar em itálico, use: <i> (NÃO SEMANTICO)
    • Para colocar em itálico, use: <em> (SEMANTICO)

  • Sublinhado:
    • Para colocar um texto em sublinhado, use: <u> (NÃO SEMANTICO)
    • para colocar um texto em sublinhado, use: <ins> (NÃO SEMÂNTICO)

  • Riscado ao meio (deletado):
    • Para colocar um texto riscado no meio, use: <del>

  • Sobrescrito
    • Para colocar um texto sobrescrito, use: <sup>

  • Subscrito
    • Para colocar um texto subscrito, use: <sub>

  • Marcador de Texto:
    • Para marcar um texto (igual marcador de texto), use: <mark>

  • Formatação para escrever código-fonte (melhor):
    • Para escrever códgio-fonte de maneira mais nítida, use: <code>

  • Abreviações:
    • Para usar Abreviações, use: <abbr title="[Texto]">

Quebra de linha (Pular linha):

Para quebrar linhas, use: <br>

Linha de Divisão:

Para dividir um conteúdo com uma linha, use: <hr>

Símbolos especiais:

<, >, ®, ©, ≶, ⌖, ™, ▵, €, £, ℜ, ¥, ¢, Δ, ↑

OBS: Tem no material da aula.

Citação:

Para citar a fala de álguem, use: <q>

OBS: Pode usar também o "[Texto]", mas é melhor o <q>, pois dá sentido e pode ajudar em estilos do CSS.

Citação em Bloco:

Para citar em bloco, use: <blockquote cite="[link]">

Espaçamento da linha das frases:

Para espaçar as linhas das frases, use: line-height: [parâmetro];
